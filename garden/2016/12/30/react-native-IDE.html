React Native 开发环境搭建

30 Dec 2016

教你如何搭建 React Native 开发环境，以 Android 为例。

  • React Native
  • 开发环境搭建
  • 虚拟机
  • 模拟器

安装 node.js 下载地址

安装 npm

curl -L https://npmjs.com/install.sh | sh

安装 react-native-cli

npm install react-native-cli -g

安装 JDK 下载地址

安装 Android Studio 下载地址

  • 选择 Custom

  • 勾选 Performance、Android Virtual Device

  • 选择 Configure -> SDK Manager

    • 选择 SDK Platforms -> 勾选 Show Package Details -> 勾选 Android 6.0 (Marshmallow) 中 Google APIs、Android SDK Platform 23、Intel x86 Atom System Image、Intel x86 Atom_64 System Image、Google APIs Intel x86 Atom_64 System Image

    • 选择 SDK Tools -> 勾选 Show Package Details -> 勾选 Android SDK Build Tools 中 Android SDK Build-Tools 23.0.1

设置环境变量

# 创建 .bash_profile 文件并编辑
sudo vi ~/.bash_profile   

# .bash_profile 文件添加如下两行
export ANDROID_HOME=~/Library/Android/sdk
export PATH=~/Library/Android/sdk/platform-tools/:$PATH

# 生效 .bash_profile 文件
source ~/.bash_profile

安装 VirtualBox 下载地址

安装 Genymotion 下载地址

  • 注册账号 注册地址

  • 添加 Virtual Device

  • 设置 ADB

    • 选择 Use custom Android SDK tools -> 填写 /Users/[用户名]/Library/Android/sdk/

  • 创建项目并启动

react-native init app
cd app
react-native run-android
